Lattice

A learning companion that explains any concept and shows it. For a question like “How does CRISPR work?”, Lattice generates a personalized written explanation and a fitting visualization. The core is an LLM “director”: instead of hand-coding each visual, a model emits a small declarative spec (actors + a timeline, or a particle / parametric scene description) that a single renderer plays — so it generalizes across concepts and domains.

Three rendering paths cover most concepts: process animations (an eased, camera-driven 2D player that shows mechanisms step by step), particle scenes (glowing point clouds for things that are points — galaxies, strange attractors), and parametric 3D models (real geometry for structures, e.g. the Colosseum). The three pieces below run live, right on this page.

Anime Goods Agent
